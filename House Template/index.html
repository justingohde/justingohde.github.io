Computer Science I

An basic website demo by J. Gohde

 

This demo has two awesome main sections:

  • Static House - Demonstrates how to include a static picture on a webpage
  • Dynamic House - Demonstrate how to include a Processing sketch on a webpage

Each of these webpages uses two resource files:

  • "style1.css" for organizing global colors, fonts, etc.
  • "menu.js" for generating the same menu for every page in thse site
